  • TNConfiguration

    A configuration class that can be used with TNEnvironment, TNRouteConfiguration and TNRequest. A configuration object follows the following rules:

    1. When a TNConfiguration object is passed to a TNEnvironment, each TNRouter (with its routes) will inherit this configuration.
    2. When a TNConfiguration object is passed to TNRouter, all its routes will inherit this configuration.
