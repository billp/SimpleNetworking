TNConfiguration

public class TNConfiguration
extension TNConfiguration: NSCopying

A generic configuration class that can be used with TNEnvironment, TNRouteConfiguration and TNRequest. If a TNConfiguration is passed to a TNEnvironment, each TNRequest will inherit this configuration. Also, each request can have its own TNConfiguration whose settings will override those from environment.
